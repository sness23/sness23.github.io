<h3 class="title">pdbset</h3>

<p>to convert a pdb file into a .pir file with the standard one letter sequence:</p>

<pre>
pdbset XYZIN $1.pdb XYZOUT coords/$1_out.pdb << END > log/$1.log
EXCLUDE HOH
SEQUENCE SINGLE $1.pir
END
</pre>
