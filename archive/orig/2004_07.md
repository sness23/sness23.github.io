<h3 class="title"><a href="http://www.nytimes.com/2004/07/13/science/13magn.html?pagewanted=2">nyt artickle</a></h3>

<p>Like many of the Earth's invisible rhythms, the field reversals are typically slow, taking anywhere from 5,000 to 7,000 years to complete.</p>

<p>Strong evidence of their reality first emerged in the 1950's and 1960's when scientists towing magnetic sensors behind ships found that the rocky seabed exhibited odd stripes of magnetization.</p>

<p>It turned out that continuous flows of seabed lava became alternately magnetized over the ages as the polarities of the Earth's field switched one way, then the other. The seabed acted like a huge tape recorder, and the same proved true of the layered deposits of old volcanoes on land.</p>

<p>How did the rocky memories form? Molten lava proved to hold tiny mineral grains that acted like innumerable compasses, or miniature magnets, freely aligning themselves with the contemporary field. But as the lava cooled, the tiny compasses froze in place, immobile even if the field shifted. Experts called it paleomagnetism and found that the tiny compasses were often made of magnetite, a naturally magnetic mineral.</p>

<p><a href="http://www.psc.edu/~dnave/notes/w2kvmw.phtml">sweet</a> - instructions for getting vmware to run off a non-bootable cdrom.</p>

<pre>
You are here: www.psc.edu/ D. Nave @ psc.edu/ D. Nave's Notes/ Installing Win2K from CD in VM-Ware 3.2 for Linux 

<p>To install Windows 2000 Professional in VM-Ware 3.2 from non-bootable CD: <br />
Initialize a new VM-Ware config for Windows 2000 Professional via the configuration wizard. Point the CD-ROM device at its proper /dev location, but point the floppy at a file (e.g. /vmware/win2000Pro/floppy). VM-Ware will warn that this "device" doesn't exist. This is fine; continue anyway. Make sure the box is checked to enable (connect) the floppy at Power On. <br />
When the wizard is complete, select Settings -> Configuration Editor from VM-Ware's menu, and edit the Floppy Drives entry. Change the combo box from Device to File, and make sure that the path is the same as in (1). <br />
Mount your CD-ROM under Linux, and find cdboot1.img on the mounted file system. Copy cdboot1.img into the floppy's file (e.g. from (1) above). <br />
Power on VM-Ware. This should bring up the normal Windows 2000 Disk 1 installation screen. If not, make sure that: <br />
the Floppy configuration from Settings -> Configuration Editor points at the right file;<br />
that you've copied cdboot1.img into the file indicated below the combo box; and,<br />
that the floppy is connected at Power On (the corresponding check box should be darkened);<br />
When Windows 2000 Setup asks for a new Disk N, copy the corresponding cdbootN.img into the floppy file.<br />
That's it. Windows 2000 Setup should proceed as normal.<br />
Démian Nave @ PSC \\ Last updated 07:00pm 12/31/69, PHP V3.0.18</p>
